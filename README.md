# 🦉 Owlnest
**Never lose your flow again.** Owlnest is a smart context-saver that helps you track your work, recall past sessions, and capture notes effortlessly.

## ✨ Features
*   **Application Tracking**: Automatically remembers which applications you were using.
*   **Smart Recall**: Quickly jump back to previous work progress with search and history.
*   **Seamless Note-taking**: Capture progress instantly with our built-in capture modal that triggers when application session ends.
*   **Privacy First**: Built as a native desktop application to keep your productivity data where it belongs—on your machine.

## 🚀 Getting Started
1.  Go to the [Latest Releases](https://github.com/ascxiao/owlnest-releases/releases/latest) page.
2.  Download the `Owlnest_x.x.x_x64_en-US.msi` (for Windows) or the Setup EXE.
3.  Run the installer and follow the on-screen instructions.
4.  Launch Owlnest from your Start Menu.
